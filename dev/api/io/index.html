Input / Output

NetCDF output

Pelagos.Output.write_snapshot — Method
write_snapshot(filename, state, grid_info, time_yr)

Write an ocean state snapshot to a NetCDF file.

Arguments

  • filename : output file path (.nc)
  • state : NamedTuple with fields: T, S, u, v, w, psi_bt, rho, pressure
  • grid_info : NamedTuple with: lon, lat, zcentres, zfaces
  • time_yr : model time in years (written as a coordinate)

Pelagos.Restarts.write_restart — Method
write_restart(filename, T, S, time_yr)

Write a restart file containing tracer fields and the current model time.

Arguments

  • filename: path to output .nc file
  • T : temperature (nlon, nlat, nz), °C
  • S : salinity (nlon, nlat, nz), psu
  • time_yr : current model time, years
